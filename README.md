## Getting Started

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd frontend-test-app-for-shai
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start the development server**
   ```bash
   pnpm dev
   ```

4. **Run tests**
   ```bash
   # Run tests in watch mode
   pnpm test

   # Run tests with coverage
   pnpm test:coverage
   ```

5. **Build for production**
   ```bash
   pnpm build
   ```
